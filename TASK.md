# Dashboard Telemetrica AI-Hardware

**Categoria**: automazione
**Difficoltà**: Facile
**Stima**: 1-2gg

## Descrizione

Web dashboard in Node.js + Grafana/Lite che aggrega dati da systemd journal, nvidia-smi (P40+3050), ventole P40 (via script custom) e carico CPU. Push di alert su Telegram se VRAM >85% o temp P40 >75°C. Integrazione diretta con heartbeat script.
